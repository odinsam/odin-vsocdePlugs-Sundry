**简介:**

代码统计插件:
命令:sundry.codelinestatistics.statistics

参数:
代码提交人:需要统计的代码提交人
开始时间:代码统计开始时间
结束时间:代码统计结束时间,如果没有则为当前日期

输出:
第一个返回为 当前提交人在当前仓库所有的代码提交情况
第二个返回为 当前提交人在当前仓库时间段内的代码提交情况
